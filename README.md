## 💡 核心特色：我们到底有什么不一样？

### 1️⃣ 问古笺 · 你的千年笔友已上线
> 「有问题？写信给古人啊！」

- 输入你的困惑，AI自动匹配最相关的古籍主题
- 三轮「书信往还」，像真的在跟古人笔谈
- 每封「古人回信」400-500字，有问候、有典故、有金句、有建议
- 引用全部标注出处，拒绝「古人说过」式的瞎编 🚫

### 2️⃣ 今朝录 · 古人的日常打开方式
> 「看看千年前的今天，大家都在忙啥？」

- 黄历宜忌：不是封建迷信，是古人的生活仪式感 ✨
- 今日心境：你的情绪，古人都懂
- 十二封「与君书」：按月令节气，慢慢读慢慢品
- 推荐音方：古乐疗愈，比白噪音还上头 🎵

### 3️⃣ 梦华笺 · 周公解梦遇上弗洛伊德
> 「昨晚做的梦，到底啥意思？」

- 中国传统视角：周易八卦、五行、紫微斗数、奇门遁甲…（不是算命，是文化！）
- 现代心理学视角：弗洛伊德、荣格、潜意识分析…（中西合璧，疗效翻倍）
- 结果也是「一封来信」的格式，解梦也能很有氛围感 🌙

### 4️⃣ 藏笺阁 · 你的私人古籍博物馆
> 「收藏的不是信，是某次被戳中的心情。」

- localStorage本地存储，你的秘密只有你知道 🔒
- 收藏的回信、记录的心情、做过的梦，全都在这
- 时不时翻一翻，发现原来自己已经成长了这么多

## 🎯 行业痛点：我们来解决这些「老大难」

### 痛点一：传统文化「出圈难」，年轻人「入坑难」
> 现状：博物馆人满为患，但《论语》翻开第一页就合上了。

**我们的解法：** 用「写信」这种浪漫又熟悉的方式，把古籍智慧包装成年轻人愿意读、读得懂、用得上的内容。国风的皮，智慧的魂，不端着，不装逼。

### 痛点二：AI内容「一本正经地胡说八道」
> 现状：问AI要几句古文名言，它能给你编出十句「鲁迅说过的话」。

**我们的解法：** 所有古籍引用均来自经过整理的真实文本，每一条都标注篇目和出处。宁少勿滥，宁缺毋滥。毕竟——**真实，才是对传统文化最大的尊重。**

### 痛点三：知道很多道理，依然过不好这一生
> 现状：收藏了100篇国学文章，第二天还是老样子。

**我们的解法：** 每封回信都附一条「今日小事」——不是空洞的大道理，是今天就能做的具体行动。从「读一首诗」到「给朋友发条问候」，微小但确定。

### 痛点四：传统文化产品「老龄化」严重
> 现状：界面像上个世纪的网站，交互比做文言文阅读还费劲。

**我们的解法：** 新中式美学设计，手机电脑都适配。用年轻人喜欢的方式，讲年轻人该听的故事。

## 🛠 技术栈

- **HTML5** — 结构的脊梁骨
- **CSS3** — 颜值担当（新中式美学，我们是认真的）
- **原生JavaScript** — 不整花里胡哨的框架，性能拉满
